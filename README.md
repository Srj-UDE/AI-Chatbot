# 🤖 AI Chatbot

A dialog-based AI chatbot developed in Python using LangChain and OpenAI chat models, deployed through a Streamlit web interface. The chatbot answers user queries purely based on the knowledge of the language model, without using external data sources or retrieval mechanisms such as Retrieval-Augmented Generation (RAG).

This project demonstrates how modern LLM frameworks can be used to build simple conversational AI systems with an intuitive web interface.

---

## 🚀 Features

- 💬 Interactive AI chatbot interface
- ⚡ Powered by OpenAI chat models
- 🔗 Built using LangChain
- 🌐 Web interface created with Streamlit
- 🧠 Responses generated purely from model knowledge
- 🔑 Secure API access via OpenAI API key

---

## 🏗 Project Architecture

```
User Input (Streamlit UI)
        │
        ▼
LangChain Chat Pipeline
        │
        ▼
OpenAI Chat Model
        │
        ▼
AI Generated Response
        │
        ▼
Displayed in Streamlit Interface
```

## 📦 Dependencies

The project uses the following Python packages:

```
langchain
streamlit==1.53.1
langchain-openai==0.3.18
dotenv==0.9.9
pip==25.2
```

Install the dependencies with:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Setup

Create a `.env` file in the root directory of the project:

```
OPENAI_API_KEY=your_openai_api_key_here
```

The application uses this key to access the OpenAI chat model.

---

## ▶️ Running the Application

Start the Streamlit application with:

```bash
streamlit run chatbot.py
```

After running the command, open the local address shown in the terminal (usually):

```
http://localhost:8501
```

You can then interact with the chatbot through the web interface.
